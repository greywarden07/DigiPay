How It Works

DigiPay makes it easy to receive UPI payments from anyone with a few simple steps:

  1. Enter your UPI ID (VPA) and payment details
  2. Generate a QR code with your information
  3. Ask others to scan the QR with any UPI app
  4. Receive money directly to your linked bank account

UPI Parameters

The app supports the following UPI parameters:

Parameter Description Required
payeeVpa (pa) UPI Virtual Payment Address (e.g. merchant@bank) Yes
payeeName (pn) Name of the merchant or individual Yes
amount (am) Payment amount No
transactionRef (tr) Unique transaction reference No
transactionNote (tn) Note displayed to the payer No
currency (cu) Currency code (default: INR) No
